# 📚 microservice_books - Organize your personal library with ease

## 📖 About this application

Microservice_books manages your digital book collection using a modern microservice structure. This software organizes your reading list and provides a smooth interface for your Burmese language titles. It tracks what you read and helps manage your library data without complex setup.

## 💻 System requirements

* Operating System: Windows 10 or Windows 11
* Processor: Intel Core i3 or equivalent
* Memory: 4 GB RAM
* Storage: 200 MB of space
* Network: Connection to the internet for the first setup

## ⬇️ Setup and installation

Follow these steps to put this software on your computer.

1. Go to the [official release page](https://github.com/Akashkashyap11/microservice_books/releases).
2. Look for the latest version listed at the top.

## ✨ Key features

* Library management: Group your books by author, category, or title.
* Burmese language support: Search and view books with Burmese character support.
* Fast performance: The architecture handles large lists without slowing your computer.
* Simple interface: Navigate through your library with clear buttons and lists.
* Data privacy: All library data stays on your local machine.

## 🛠️ How to use the software

Start by opening the application from your desktop. You will see a home window that lists your books.

### Adding a new book
Click the plus button at the top corner of the screen. A small window appears. Type the title of the book in the first box. Type the author name in the second box. Click save. The book appears in your list immediately.

### Searching your library
Type any word into the search bar at the top of the interface. The list filters automatically as you type. If you search for a title in the Burmese language, the software updates the list to show matches for those specific characters.

### Managing metadata
Right-click on any book in the list. Choose edit to change information about the book. You can update the cover image, the year of release, or the status of your reading progress. Press save to apply the changes.

## 🛡️ Data management

The program creates a folder in your documents directory to keep your library safe. This folder contains a library file that holds all your information. You should back up this folder once a month to a flash drive or cloud storage to prevent data loss.

## 🧩 Troubleshooting common issues

If the application does not start, verify you have the latest updates for Windows.
If the search results look empty, clear the search bar by clicking the small x icon.
If you see an error message during installation, check that you have enough space on your hard drive. 
If the text does not show the Burmese script correctly, check your Windows language settings to make sure your system supports these characters.
